OrbitScope: User Documentation

Features

  • Interactive 3D planet orbit simulation
  • Learning mode with zoomed-in canvas for each planet
  • Dynamic facts loaded from JSON
  • Sound effects and PWA install support

How to Use

  1. Click “Launch Learning View” to enter canvas mode
  2. Select a planet to learn about it with image, facts, and sound
  3. Scroll down to view full solar system simulation

Installation

Open the app in a supported browser like Chrome or Edge. Use the “Install” button or browser menu to install as a PWA.
